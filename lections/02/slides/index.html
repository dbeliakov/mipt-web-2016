What is CSS

  • Cascading Style Sheets
  • Содержит информацию о внешнем виде
  • Представляет собой набор правил
  • Правила распрастрониются на объекты, отобранные селекторами
  • CSS включается в HTML страницу через атрибут style , <style> or <link>
a {
	color: blue;
}

CSS selectors

  • Стандарт
  • Виды селекторов:
    • Type selector a {}
    • Universal selector * {}
    • Attribute selector [extra~=style] ... <span extra="style other">
    • Class selector .highlight {} ... <span class="highlight">
    • ID selector #special {} ... <span id="special">
  • Могут содержать другие селекторы при помощи комбинаторов

Combinators

  • selector1,selector2 is like "or"
  • selector1 selector2 is for descendants
  • selector1 > selector2 is for children (direct descendants)
  • selector1 + selector2 is for direct siblings
  • selector1 ~ selector2 is for general siblings

Pseudo-classes

  • Dynamic
    • :hover - mouse is over
    • :focus - element is focused
    • :visited - link was previously visited
  • Structural, e.g. :nth-child
  • :target
  • :lang
  • UI states: :enabled, :disabled, :checked, :indeterminate
  • :not

Pseudo-elements

  • ::before, ::after insert elements with special option "content"
  • ::first-letter, ::first-line
  • ::selection - highlighted text
a::after {
  content: attr(hint);
  color: red;
}
<a hint="test"></a>

CSS units

  • Absolute: px, mm, cm, in, pt, pc
  • Font: em, ex, ch, rem
  • Viewport: vh, vw, vmin, vmax
  • %
  • auto

CSS box model

CSS box size

  • width, height - give a hint to a browser
  • max-width, min-width, max-height, min-height - strongly demand

CSS box model options

  • margin; margin-left, margin-top, etc. May be negative.
  • border: border-style, border-width, border-color, etc.
  • padding: padding-left, padding-top, etc. May not be negative.
div#container {
  margin: 10px 1.5em 1% 2vh;  /* top right bottom left */
  border: 2px solid blue;
  padding: auto 3vmin;  /* top = bottom, right = left */
}

CSS layout

display

Sets which element box type is used. Documentation.

  • none - invisible, do not take part in the layout
  • block - no horizontal elements before and after
  • inline - like text
  • inline-block - inline box
  • flex - modern grid
  • table* - traditional grid
  • ...

flex-box

			display: flex;
  • Documentation
  • Статья на Habrahabr
  • flexboxgrid - CSS framework

position

Задает, как элементы позиционируются MDN, Standard

  • static - this is the default, obey the normal flow
  • relative - move element without changing the layout
  • absolute - allows setting exact coordinates (left, top, etc.) in the parent element, ignored in layout
  • fixed - allows setting exact coordinates in the viewport, element is ignored in layout
  • sticky - similar to relative, read an example, not supported by Chrome

float

  • Excludes element from the flow, MDN
  • float: left; (right)

Hiding elements

  • display: none; - changes layout
  • visibility: hidden; - saves layout, but triggers the event (opposite is visible)
  • opacity: 0; - saves layout, does not trigger the event
  • clip: rect(0, 0, 0, 0); - saves layout, does not trigger the event, exotic, should not be used

Centering horizontally

{
  display: table;  /* block requires setting width */
  margin-left: auto;
  margin-right: auto;
}
{
  display: inline;  /* inline-block */
  text-align: center;
}
{  /* parent */
  display: flex;
  flex-direction: row;
  justify-content: center;
}
